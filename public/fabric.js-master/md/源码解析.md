
调用栈

```
var canvas = new fabric.Canvas('c');
createClass -> klass -> initialize // 初始化方法
|- initialize
 |- this._initStatic(el, options); //  设置lowerCanvasEl和ctx, _initOptions 初始化选项， 设置背景图片， 背景色相关
 |- this._initInteractive 
    |- _initWrapperElement 创建一个容器canvas-container包裹canvas
    |- _createUpperCanvas 创建一个upper-canvas的canvas图层放在canvas上，
    |- _initEventListeners 添加各种事件监听
    |- _initRetinaScaling 将canvas放大1.25倍
|- _createCacheCanvas 创建一个canvas对象作为缓存容器

```
